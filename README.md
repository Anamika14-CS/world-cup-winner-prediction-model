About the Project
Cricket is a game influenced by many factors such as team performance, consistency, past records, and match conditions.
The aim of this project is to analyze past World Cup data and use machine learning to predict which team has a higher chance of winning.
This project helped me connect theory with practical implementation instead of just learning algorithms on paper.
